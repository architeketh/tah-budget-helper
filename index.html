Supabase Autosave

The app can autosave to Supabase in the background and sync across devices using one shared budget ID and sync code.

Use the same budget ID and sync code on desktop and mobile to keep them in sync.

Why this setup is the simplest fit

A single `index.html` still works well on GitHub Pages, stays easy to update, and now autosaves to Supabase for cross-device syncing. The backup button is still there if you want an offline copy.

Monthly reset made easy

Use Start Next Month to carry recurring bills forward with paid boxes cleared. One-time items stay behind unless you choose to copy everything.
